About • RIMAN

RIMAN은 글로벌 뷰티 & 웰니스 브랜드로, 단기적 효과보다 지속 가능한 변화에 가치를 둡니다. 말레이시아 커뮤니티에서는 보통 RIMAN, 라인업으로는 Incellderm 또는 ICD라는 키워드가 통용됩니다.

About • Incellderm (ICD)

Incellderm(인셀더姆/ICD)은 RIMAN의 대표 스킨케어 라인으로 피부에 자극을 줄이면서도 밀도 있는 관리 루틴을 제안합니다. 제품 선택은 현재 피부 컨디션과 생활 리듬에 맞게 가볍게 시작하세요.

How to Join (Malaysia)

  1. 공식 사이트 접속 · kr.riman.com (또는 지역 페이지)
  2. 회원가입 단계에서 Referral Code 입력
  3. 정상 등록 후, 커뮤니티 가이드를 참고하여 천천히 시작

말레이시아에서는 보통 Referral Code 또는 Referral ID라는 표현을 사용합니다.
